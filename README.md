# gofs-webui

## Installation

The first need [Go](https://go.dev/doc/install) and [Node](https://nodejs.org/en/download) installed, then you can use
the below command to install `gofs-webui`.

On Windows

```bat
$ scripts\build.bat
```

On Linux

```bash
$ chmod +x ./scripts/build.sh && ./scripts/build.sh

```
